What this repo is

This repository contains the PlexCode IDE, the Audax UI Builder, and the NCOM SDK. Here’s a reminder:

  • .plx files → PlexCode scripts.
  • .mf files → Manifesto files. UI-enabled ones trigger Audax.
  • .TAG files → Signatures, accounts, and authentication info.
  • NCOM DEV NOTES/ → contains repo tree, notes, and HTML guides.

How to use it

Start by opening the PlexCode IDE:

  • Create new .plx or .mf files.
  • Type build~ UI in a .mf to launch the Audax UI editor.
  • Use Store~ commands to save state, auto-corrects to Store | File-Name ["filename.mf"].
  • Use Show~ ALL | Tree to see the live repo file tree in the IDE.
